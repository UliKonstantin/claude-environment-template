# Claude environment template

Starter layout for projects that use Claude (Cursor) with consistent rules and context.

## What's included

- **`.claude/`** — Project rules, commands, reference docs, and PRD/settings.
- **`Claude.md`** — This file: project overview and conventions (customize per repo).
- **`.agents/`** — Plans and agent-related artifacts (e.g. `.agents/plans/`).

## How to use

1. Clone or copy this repo as the base for a new project.
2. Replace this `Claude.md` with your project’s overview, stack, structure, and commands.
3. Edit `.claude/` as needed (PRD, commands, reference, settings).
4. Use `.agents/plans/` for feature plans and execution notes.

## Reference

- [Cursor rules](https://docs.cursor.com/context/rules-for-ai) — How rules and `Claude.md` are used.
- [.claude/](.claude/) — Where to put project-specific instructions and docs.
